# Diff b/w let , var and const :-

1). var : the varible made with var is global scope and functional scope like we can use its value anywhere in code. Cons about it is that we can create the same variable name several times and store different values but js engine will remind only last declared variable (it can create biggest problem in code).

2). let : the variable made with let is block scope. If we assign a variable with let then we could not create the same varible name next time it will through an error .But one thing about this variable is that we can change its value several times .

3). const : the variable made with const is also block scope but we can not change or assign its value because it is constant .
